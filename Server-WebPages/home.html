Welcome to My Server

About This Server

This server is a result of my journey into understanding server administration, networking, and web hosting. It runs on an Ubuntu virtual machine hosted using Oracle VirtualBox with a bridged network setup for seamless integration with my local network.

How I Built This Server

  • Configured an Ubuntu server on a virtual machine.
  • Set up a bridged network for better accessibility.
  • Enabled SSH for remote access and installed Nginx for serving web pages.
  • Handled network challenges, including dynamic IP and firewall configuration.
  • Configured this server to serve multiple websites and ensured it can scale in functionality.

My Learnings

This project taught me:

  • The fundamentals of server setup and management.
  • Networking concepts like IP addressing and port forwarding.
  • Web hosting basics, including Nginx configuration.
  • Overcoming real-world challenges such as limited network resources and ensuring security.
